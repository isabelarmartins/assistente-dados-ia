# 🤖 Assistente Inteligente de Análise de Dados com IA

Este projeto é um assistente que permite realizar consultas em linguagem natural sobre um banco de dados, utilizando Inteligência Artificial para gerar automaticamente queries SQL.

---

## 🚀 Funcionalidades

- 💬 Conversão de linguagem natural em SQL
- 🧠 Integração com modelo de IA
- 📊 Execução automática das queries no banco SQLite
- 📈 Exibição dos resultados em tabela e gráfico
- 🔍 Visualização da SQL gerada (modo expandido)
